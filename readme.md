# FINANCIAL WALLET

## Description

This project is a personal wallet for making money transactions.

#### Technologies

- TypeScript
- NodeJS

## How To Use

### Installation
- Root folder run

```html
    yarn install
```
- Run migrations
```html
    yarn typeorm migration:run
```
### Run
```html
    yarn dev:server
```
